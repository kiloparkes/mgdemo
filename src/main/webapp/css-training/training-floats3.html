Notice that the background image for the left side bar is not as long as the main text. It stops short. This is because a box is as long as the containing text unless you specify a height. Therefore, even though we have repeat-y for the image it only goes as far at the height of the containing text. The solution is to wrap the side bar and content and add the image to the wrapper with repeat-y
